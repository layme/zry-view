# zry view

## 简介
基于Vue.js，搭配使用[iView](https://www.iviewui.com) UI组件库开发的一套后台管理页面

## Install
```bush
// install dependencies
npm install
```
## Run
### Development
```bush
npm run dev
```
### Develop(Build)
```bush
npm run develop
```
### Test(Build)
```bush
npm run test
```
### Prepare(Build)
```bush
npm run prepare
```
### Production(Build)
```bush
npm run build
```
